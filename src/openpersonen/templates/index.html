Open Personen biedt een implementatie van de Haal Centraal BRP bevragen API specificatie en kan verbinden met een StUF-BG koppeling voor het ontsluiten van persoonsgegevens. Ook kan Open Personen eenvoudigweg StUF-BG berichten converteren naar BRP API berichten zonder daadwerkelijke verbinding. Ten slotte kan Open Personen gebruikt worden met een testset die u kunt importeren of zelf aanmaken via de gebruikers interface.

Open Personen voorziet in de behoefte van verschillende andere componenten om persoonsgegevens op te vragen en personen te koppelen (bijv. aan een zaak in de Open Zaak ).
